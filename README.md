Java 3d simple engine (j3dsengine).

This project uses:

    - Lightweight Java Game Library 3 (LWJGL3 OpenGL, OpenAl and more for java)
    
    - Open Dynamics Engine (ODE4J Physics engine)
    
    - Json library
